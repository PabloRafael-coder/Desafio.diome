Como surgiu a Internet?

Na década de 1950 ocorreu a guerra fria, disputa entre os Estados Unidos e a União Soviética, estes dois países desenvolveram várias tecnologias. O lançamento do Sputnik (satélite) lançado pela União Soviética fez com que os Estados Unidos criassem a ARPA (Advanced Research Projects Agency) como forma de concentrar melhores tecnologias frente a União Soviética.

Então criou-se a rede de comunicação que ficou conhecida como ARPAnet, foi utilizada para comunicar-se com segurança e sigilosidade, ou seja, garantiriam que as documentações desenvolvidas não seriam interceptadas pelos inimigos, posteriormente começou a surgir diversas redes de comunicações, porém não teria como estabelecer-se uma comunicação global.

A chegada do protocolo TCP/IP (Transmission Control Protocol/Internet Protocol) até hoje é implementado na internet, substituiu todas redes de comunicações e virou o principal meio de comunicação utilizado pela internet.

O que é HTML?

O HTML (Linguagem de Marcação de HiperTexto) é um arquivo de marcação que possibilita criar e estruturar páginas de Web. É uma linguagem de texto simples, portanto, não é uma linguagem de programação. Os desenvolvedores podem utilizá-la para construir uma web, podem utilizar diversos arquivos, como áudios, vídeos e imagens, além de textos.

O que é uma estrutura Cliente-Servidor?

Um servidor é configurado para armazenar documentos, aplicações e prestar serviços que são capazes de serem requisitados, estas requisições são feitas pelos clientes por meio das tecnologias que podem ser computadores, celulares e tablets. Quando a internet é acessada pelo navegador, geralmente, é feita uma requisição na web, podendo ser as redes sociais, como: facebook, instagram, netflix, twitter ou websites como blogs, entre outros. Todas essas informações ficam armazenadas em servidores que geralmente ficam em datacenters. Para uma melhor compreensão, temos:

  • Cliente: Requisita o conteúdo na Web
  • Internet: Meio utilizado para fazer a requisição
  • Servidor: Disponibiliza o conteúdo requisitado

O que é marcador e como estruturar um texto?

Os marcadores (tags) são elementos que ajudam a delimitar e identificar os textos. São construídos no formato e cada tipo tem finalidade específica. Ao estruturar um texto em HTML deve-se utilizar algumas tags importantes, exemplos:


TAG < P > : Essa tag é utilizada para separar blocos de parágrafos, ajudam a estruturar o texto.
TAG < H >: Utilizada em títulos e subtítulos, pode ser atribuído um valor de 1 a 6, sendo o número 1 o título e os demais subtítulos.
Exemplos: < h1 >, < h2 >, < h3 >, < h4 >, < h5 > e < h6 >.
TAG < UL >: Criar uma lista Não ordenada, sem numeração ou letras.
TAG < OL >: Criar uma lista Ordenada, atribuindo valores.
TAG < LI >: Define cada item da lista.
TAG < A >: Essa tag permite criar um hiperlink, sendo navegável pela web.
